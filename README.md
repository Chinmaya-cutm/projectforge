# ProjectForge

![React](https://img.shields.io/badge/React-19-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6-yellow)
![Vite](https://img.shields.io/badge/Vite-Latest-purple)
![Status](https://img.shields.io/badge/Status-Active-success)

A modern software planning workspace built for developers to manage software projects from idea to implementation. ProjectForge combines project planning, feature tracking, roadmap management, task organization, note-taking, database design, and analytics into a single platform.

## Live Demo

https://projectforge-bay.vercel.app

## Features

### Project Management

* Create and manage multiple software projects
* Organize project information in dedicated workspaces
* Persistent data storage using LocalStorage

### Feature Tracking

* Add and manage project features
* Track feature completion progress
* Automatic project progress calculation

### Task Management

* Organize tasks using Todo, In Progress, and Done states
* Monitor development workflow
* Track project execution efficiently

### Roadmap Planning

* Create project phases and milestones
* Track roadmap progress
* Plan software releases

### Notes System

* Store project documentation
* Save implementation details and ideas
* Maintain development notes

### Database Designer

* Create database tables
* Add and manage table columns
* Visually design project database structures

### Analytics Dashboard

* Project statistics overview
* Feature completion metrics
* Task completion analytics
* Development progress insights

### Search & Export

* Instant project search
* Export complete project data as JSON
* Local project backup support

## Installation

Clone the repository:

```bash
git clone https://github.com/Chinmaya-cutm/projectforge.git
```

Navigate to the project directory:

```bash
cd projectforge
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
